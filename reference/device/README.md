# Device Reference RBCP Implementations

This document contains links to reference implementations of RBCP devices (emulated ROMs).

- [One ROM](https://github.com/piersfinlayson/one-rom/tree/main/plugins/user/host-control): One ROM is a flexible replacement ROM for 24, 28, 32 and 40 pin ROMs, based on the RP2350 microcontroller.  It's `host-control` plugin implements RBCP and can be used as a reference implementation of an RBCP device.
